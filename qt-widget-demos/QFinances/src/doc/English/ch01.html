1. Basic possibilities

For that, to begin to work with the program it is necessary to register the account in the system, if already registered then to choose from a list necessary. The program contains four buttons-tools of bars:

  1. Add operation – serves for introduction of operations (receipts, charges) to the base.

  2. View – contains a menu which next commands are in

    • Show year transaction – to choose the year of operations.(to represent only those operations which was conducted in the chosen year)

    • Custom filter – to represent operations which answer chosen a condition.

    • Reset filter – to represent all of operations for all.

    • Cash panel – is a show/hide cash panel.

    • Mortgagors panel – is a show/hide debtors panel.

  3. Print report – print/save operations are represented in a file.

  4. Preferences – show the dialog of program and account settings.

  5. Help – is help documentation.
