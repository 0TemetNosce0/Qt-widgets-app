Installation and system requirements

For installation the program necessary is a library of Qt4, which can be loaded from http://trolltech.com. Enter the programs in a catalogue:

  1. qmake.

  2. make

  3. make install

It is necessary for successful installation of the program:

  1. To have rights for a record in a directory where the program is.

  2. Presence in the system of packages of qt4 and qt4-devel.

Important
Recommended to keep the program in the home catalogue of user in the case Linux of the system, or not system disk in the case Windows of the system. In opposite case the loss of files of these accounts is possible at reinstall programs.
