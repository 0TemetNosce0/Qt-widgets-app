Інсталяція та системні вимоги

Для інсталяції програми необхідною є бібліотека Qt4, яку можна завантажити із http://trolltech.com. Введіть в каталозі програми:

  1. qmake.

  2. make

  3. make install

Для успішної інсталяції програми необхідно:

  1. Мати права запису в директорії де знаходиться програма.

  2. Наявність в системі пакетів qt4 та qt4-devel.

Важливо
Програму рекомендується зберігати в домашньому каталозі користувача у випадку Linux системи, або не системний диск у випадку Windows системи. В протилежному випадку можлива втрата файлів даних рахунків при перевстановлені програми.
