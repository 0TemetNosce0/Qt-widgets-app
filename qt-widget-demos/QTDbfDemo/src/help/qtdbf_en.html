qtDbf 1.0

Warning !

  • Never edit a dbf file without making first a backup copy
  • Do not open and especially not save dbf files in multiuser environment.
  • The application can view memo files, but cannot edit them.
  • The dbf file structure it's read only yet.

So, let't see what we CAN do with the application :-).

The application can be set as the default dbf viewer.

  Fills the contents of selected cells with the given expression.
Enter Edits the value of the actual cell.
Same effect with double click on the cell
If the cell is memo, will show the memo string in a new window.
Ins Appends an empty record to the end of the file.
Del Deletes the records with at least a cell selected.
Ctrl + S Saves modifications in the dbf file.
  Configuration dialog with application language, icon sizes, dbf file structure and field type color chooser.
F1 Opens this window.
Ctrl + E Calculator
Ctrl + O Open dbf file.
Ctrl + F Set or reset filter
Ctrl + A Execute aggregate function

A few technical details.

  • The application reads the dbf file in an in memory SQLite database.
    So loading large files are a bit slow.
    All changes are aplied to the SQLite database, and they are written in dbf file with save option.
  • The application supports XBase, dBaseIII, IV, Foxpro 2.x, Visual Foxpro files.
  • The following field types are editable: character, numeric, currency, double, integer, float, logical.
