Предупреждение!

  • Не редактируйте dbf файл не сделав сначала резервной копии
  • Не открывайте и, особенно, не сохраняйте файлы в многопользовательском доступе
  • Приложение может просматривать memo файлы, но не позволяет редактировать их.
  • Структура dbf файла пока не редактируется.

Итак, давайте посмотрим, что мы можем сделать с приложением :-).

Приложение может быть установлено как приложение по умолчанию для dbf файлов.

Несколько технических деталей.

  • Это приложение считывает dbf файл базу данных SQLite в память.
    Так что загрузка больших файлов может быть медленной.
    Все изменения применяются в базе SQLite, и потом записываются в dbf файл при выборе операции сохранения.
  • Приложение поддерживает форматы XBase, dBaseIII, IV, Foxpro 2.x, Visual Foxpro.
  • Следующие типы полей могут редактироваться: character, numeric, currency, double, integer, float, logical.
