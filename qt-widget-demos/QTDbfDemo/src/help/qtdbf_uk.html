qtDbf 1.0

УВАГА!

  • Не редагуйте dbf файл без заздалегідь зробленої резервної копії.
  • Не відкривайте і, особливо, не зберігайте файли в багатокористувацькому доступі.
  • Додаток может переглядати memo файли, але не дозволяє редагувати їх.
  • Структура dbf файла покищо не редагується.

Отже, давайте глянемо, що ми можемо зробити з додатком :-).

Додаток може бути встановлено як переглядач dbf файлів за замовчанням.

  Заповнення полів заданим виразом.
Enter Редагування значень у клітинках.
Такий ефект досягається подвійним кліком по клітинці
Якщо клітинка memo типу - вміст буде відображено в новому вікні.
Ins Додавання пустих записів у кінець файлу.
Del Видалення поточного запису.
Ctrl + S Збереження змін у dbf файл.
  Конфігураційний діалог для вибору мови інтерфейсу додатку, кодування dbf файлу, розміру іконок, перегляду структури dbf файлу і призначення кольорів для типів полів.
F1 Відкриття цього вікна.
Ctrl + E Запуск калькулятора.
Ctrl + O Відкрити dbf файл.
Ctrl + F Встановити або зняти фільтр.
Ctrl + A Виконати агрегатну функцію

Декілька технічних деталей.

  • Цей додаток зчитує dbf файл в базу данных SQLite в пам'ять.
    Так що завантаження великих файлів може бути повільним.
    Всі зміни застосовуються в базі SQLite, і потім записуються в dbf файл при виборі операції збереження.
  • Додаток підтримує формати XBase, dBaseIII, IV, Foxpro 2.x, Visual Foxpro.
  • Наступні типи полів можуть редагуватися: character, numeric, currency, double, integer, float, logical.
